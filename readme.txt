== Installation ==

1. Upload the `eneacodes-faq-schema` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to any post and find the "EneaCodes FAQ Schema" meta box
4. Start adding FAQs to your posts!

== Frequently Asked Questions ==

= What is Schema.org markup? =
Schema.org is a structured data vocabulary that helps search engines understand your content better. FAQ Schema helps Google display your FAQs directly in search results.

= Will this plugin slow down my website? =
No! The plugin is lightweight and optimized for performance. All styles and scripts are loaded only when needed.

= Do I need coding skills to use this? =
Absolutely not! The plugin provides a simple, intuitive interface in your WordPress post editor.

= Can I customize the FAQ design? =
Yes! The plugin includes a beautiful, modern design that's fully responsive. You can also add custom CSS if needed.

= Is this plugin translation ready? =
Yes, the plugin includes translation files and is ready for localization.
